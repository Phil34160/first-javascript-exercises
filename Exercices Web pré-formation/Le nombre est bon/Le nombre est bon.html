Règle du jeu


Bonjour,

nous allons jouer au jeu du Nombre est bon !

Le principe consiste à trouver le nombre

selectionné au hasard par le computer

en seulement 10 essais. Sur chaque tentative,

si le nombre recherché n'est pas trouvé,

le computer renvoi comme seule indication :

"Plus haut" ou "Plus bas".

Le nombre est bon
